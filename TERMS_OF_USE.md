## 2. Descrição do Software

O Trix Audio Converter é um conversor de áudio de desktop que utiliza:

- **Backend:** Rust (axum, tokio, tao/wry)
- **Frontend:** React + TypeScript
- **Motor de conversão:** FFmpeg (utilizado sob a licença LGPL/GPL)

O Software permite conversão de arquivos de áudio entre diversos formatos, aplicação de efeitos, edição de metadados e outras funcionalidades relacionadas a processamento de áudio.
